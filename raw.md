# Data Sources

All datasets used in this project are freely available:

| Dataset | URL | Instructions |
|---|---|---|
| GDP Per Capita | data.worldbank.org/indicator/NY.GDP.PCAP.CD | Download CSV, all countries, 2010-2022 |
| Annual Rainfall | data.worldbank.org/indicator/AG.LND.PRCP.MM | Download CSV, all countries, 2010-2022 |
| Cereal Crop Yield | fao.org/faostat/en/#data/QCL | Select Cereals Total, Yield, 2010-2022 |
| Cereals Price Index | fao.org/worldfoodsituation/foodpricesindex | Download Excel file |
| Import Dependency | fao.org/faostat/en/#data/FS | Select Cereal import dependency ratio |
| Conflict Events | acleddata.com/data-export-tool | Register free, download all regions 2010-2022 |
